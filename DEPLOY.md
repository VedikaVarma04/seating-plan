# Seating Sorter

An exam seating arrangement system for faculties and students.

## Features
- Separate login/signup for Faculty and Students
- Faculty can create exam sessions and seating arrangements
- Students can view their assigned seats
- Excel file upload for student data
- Visual seating grid

## Running Locally

```bash
cd Seating-Sorter
npm install
npm run dev
```

The app will be available at http://localhost:5000

## Tech Stack
- React + TypeScript (Frontend)
- Express + Node.js (Backend)
- Vite (Build tool)
